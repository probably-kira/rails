rails
=====

How to install using brunch:
brunch new git@github.com:probably-kira/rails.git repo
cd repo
npm install

Or simple clone and do npm install, to install npm and bower dependencies(I assume that you have bower installed)
 
 
=======
How to run:
npm start
Open the browser to http://localhost:3333


=======
How to test:
npm test
Open the browser you would like to test to http://localhost:3334


=======
how to minify:
npm run-script prod
